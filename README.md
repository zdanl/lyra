The Lyra cryptocurrency depo presents a hopefully secure and open-source platform, enabling the owner to accept and aggregate clients payments in crypto and fiat. Lyra incorporates a meticulously organized implementation of DB collections including custodial wallets generated by serverless functions in NodeJS. Notably, the configuration will allow a seamless transition to fully decentralized funding by deactivating custodial wallets. Authentication, accounts, crypto funds, and fiat are seamlessly handled by Appwrite hosted on the Lyra network or a hosting of your choice. The system will be able to deal with Web 3.0 non-custodial wallets such as Metamask. While direct cryptocurrency purchases are not currently facilitated on the platform, approval for Transak On-Ramp integration has been secured, allowing users to effortlessly acquire cryptocurrency via credit card with a single click. The platform will offer optional and self-hosted KYC, coupled with the additional security measure of two-factor authentication (2FA).
